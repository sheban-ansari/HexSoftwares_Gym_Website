#  PowerHouse Gym Website

A modern and responsive **single-page gym website** built using **HTML5, CSS3, and JavaScript**.
This project showcases gym facilities, services, trainers, and membership plans in a clean and scrollable layout.

##  Features

*  Fully responsive design (mobile-friendly)
*  Smooth navigation with scrollable sections
*  Services and Facilities showcase
*  Trainer profiles section
*  Membership plans section
*  Contact form with validation (JavaScript)
*  Hamburger menu for mobile devices
*  Modern UI with dark theme and gradient effects

---

##  Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript** – Interactivity
* **AOS Library** – Scroll animations
